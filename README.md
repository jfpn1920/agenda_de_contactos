## 👋 ¡Bienvenidos usuarios a mi proyecto! agenda de contactos

<p style="text-align: justify;">

Este proyecto consiste en desarrollar una agenda de contactos en Python que permite almacenar, buscar, mostrar y eliminar contactos utilizando un diccionario como estructura principal de datos. Cada contacto se compone de un nombre y un número telefónico, los cuales se gestionan de manera dinámica durante la ejecución del programa.

El sistema está diseñado para funcionar a través de un menú interactivo en consola, donde el usuario puede seleccionar diferentes opciones para administrar la información almacenada. Esto permite una experiencia sencilla, intuitiva y organizada, facilitando la manipulación de los datos en tiempo real.

Además, el proyecto busca reforzar la comprensión de estructuras de datos fundamentales en Python, promoviendo una correcta organización del código mediante el uso de funciones. La implementación modular permite que cada operación (agregar, buscar, mostrar o eliminar contactos) esté separada en bloques independientes, lo que mejora la legibilidad, el mantenimiento y la escalabilidad del programa.

#
### ⚙️ Funcionamiento
1. Agregar contacto  
2. Buscar contacto  
3. Mostrar todos los contactos  
4. Eliminar contacto  
5. Salir  

### 🔹 Agregar contacto
- Permite ingresar un nombre y un número telefónico.  
- Valida si el contacto ya existe antes de guardarlo.

### 🔹 Buscar contacto
Permite ingresar un nombre y muestra su número si existe en la agenda.

### 🔹 Mostrar contactos
Muestra todos los contactos almacenados en el diccionario.

### 🔹 Eliminar contacto
Permite eliminar un contacto si se encuentra registrado.

El sistema se mantiene en ejecución hasta que el usuario seleccione la opción **Salir**.
